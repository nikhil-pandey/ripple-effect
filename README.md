# Ripple Effect Solver
A solver for [Ripple Effect Puzzles](http://www.nikoli.co.jp/en/puzzles/ripple_effect.html)

## Installation

1. Clone the repository
	```bash
	git clone https://github.com/nikhil-pandey/ripple-effect
	cd ripple-effect
	```
2. Install matplotlib
	```bash
	pip install matplotlib
	```

3. Run `__main__.py`
	```bash
	python ripple-effect
	# or
	python ripple-effect/__main__.py
	```
